Spingify is a small (0.6kb) function that takes an array of inputs, runs them through a spring physics function and sends the results to a callback function.

It's a pretty simple utility that can be used to create a range of useful and fun interactions.

Basic setup

          
const spring = new Springify(
  {
    propName: "x",
    stiffness: 10,
    damping: 30,
    mass: 20,
  },
  {
    propName: "y",
    stiffness: 10,
    damping: 30,
    mass: 20,
  },
  function animationCallback(x, y) {
    // Do stuff with the x and y values in here.
  }
);
