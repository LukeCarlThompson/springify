Spingify is a tiny lil' (> 1kb) utility function for creating dynamic spring animations.

It's a bare bones but flexible utility that can be used to create a range of useful and fun interactions.

Basic setup

                
const spring = new Springify(
  {
    propName: "x",
  },
  function (x) {
    // Do something with the springy x.output value or x.velocity value in here.
    console.log(x.output);
    console.log(x.velocity);
  }
);

// This is how we update the input springify will spring to.
// Notice the property name is the same as our string passed in as propName.
spring.x.input = 500;

// Call the animate method
spring.animate();

Pass one or many objects into Springify. Each object passed in must have a 'propName' property that is a unique string. The callback function will receive an object containing spingified values for each object passed in.

Each object passed into Springify will create a property in the instance with the propName string as it's name. This is where the input to be springified is set.

We don't just use a simple 'from' and 'to' prop because we want to create a dynamic spring that can respond to user input part way through an animation.

While it can be used as a 'from --> to' animation function it also allows for more dynamic interaction.

Sailboat

            
const sailboat = document.querySelector(".sailboat");
const sailAway = document.querySelector(".sailboat--away");
const sailBack = document.querySelector(".sailboat--back");

const springySailboat = new Springify(
  {
    propName: "x",
    input: 10,
    stiffness: 10,
    damping: 80,
    mass: 50,
  },
  function(x) {
    sailboat.style.left = `${x.output}%`;
    sailboat.style.transform = `rotate(${x.velocity * -0.2}deg)`;
  }
);

sailAway.addEventListener("click", () => {
  springySailboat.x.input = 90;
  springySailboat.animate();
});

sailBack.addEventListener("click", () => {
  springySailboat.x.input = 10;
  springySailboat.animate();
});

Spider

              
const spider = document.querySelector(".spider");
const spiderArea = document.querySelector(".section--example-spider");

const springySpider = new Springify(
  {
    propName: "x",
    stiffness: 30,
    damping: 50,
    mass: 10,
  },
  function(x) {
    spider.style.transform = `translateY(${x.output}px)`;
  }
);

window.addEventListener("scroll", () => {
  springySpider.x.input =
    window.scrollY - spiderArea.offsetTop + window.innerHeight * 0.5;
  springySpider.animate();
});
